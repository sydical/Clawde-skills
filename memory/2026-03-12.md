## 📝 笔记：OpenClaw Skills 使用心得

### 一、Skills 到底是个啥？

在 OpenClaw 里，助手通过 Skills（技能）来获得新能力，每个 Skills 就是一组配置和脚本告诉助手怎么调用某个外部服务。

比如：
- 想让它搜网页 → 装搜索技能
- 想让它生成图片 → 装生图技能
- 想让它分析数据 → 装数据分析技能
- 想让它帮你收邮箱 → 装邮箱技能

OpenClaw 官方内置了一些基础能力，但真正好用的，都在社区里。

**案例**：装了多个 Skills 后，每天早上龙虾已经帮我：
- 抓取了 20+ 个信息源的行业资讯
- 整理了邮件摘要，标出需要回复的
- 生成了今天要发的朋友圈配图
- 把待办事项同步到飞书

这才是 OpenClaw 的正确打开方式。

---

### 二、安装 Skill 前，先记住这三个原则

原则 1：先想场景，再找技能
- 先问自己：我每天最耗时间的重复性工作是什么？
- 优先装：搜索类、内容生成类、生图和视频类
- 先解决最痛的点，再慢慢扩展

原则 2：装完必测试，别装了就忘
- 很多 Skills 装上之后需要配置环境变量、API 密钥
- 每装一个，立刻测试一遍
- 测试通过的，才算是真装上了

原则 3：不用的删掉
- Skills 装多了，会让 Agent 运行变慢、Token 消耗变多
- 隔段时间查询已安装 Skills 列表，卸载不使用的
- 需要的时候再装回来，反正就一行命令的事儿

**【重要安全提示】**
- 安装需配置 API 密钥、授予邮箱/文档访问权限的技能时，请务必谨慎评估其安全性和来源可靠性
- 使用前先安装 Skill-Vetter 进行技能检查
