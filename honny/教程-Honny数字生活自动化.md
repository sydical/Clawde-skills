```

```

## 六、代码实现

### 6.1 核心脚本

创建一个 Python 脚本来管理整个生命周期：

```python
# honny-lifecycle.py 主要逻辑

def run_heartbeat():
    """心跳执行：检查并执行当前时段任务"""
    state = get_state()
    time_window = get_time_window()  # morning/forenoon/afternoon/evening
    
    for task in TASKS[time_window]:
        if not is_task_done(state, task):
            execute_task(task)
            mark_done(task)
```

### 6.2 命令行接口

```bash
# 查看今日计划
python3 honny-lifecycle.py plan

# 执行流水线
python3 honny-lifecycle.py run

# 查看状态
python3 honny-lifecycle.py status
```
