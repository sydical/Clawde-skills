# OpenClaw实战｜快速安装飞书官方插件

---

姐妹们！今天要跟你们聊一个我觉得特别有意义的事儿——飞书官方终于把 OpenClaw 插件做出来了！

以前我每次写 OpenClaw 的教程，评论区最多的问题就是："安装好难啊""飞书配置看不懂""能不能简单点？"

现在！终于！有解！决！了！

---

## 以前接飞书有多劝退？

说真的，之前让 OpenClaw 接入飞书，那流程能把普通人劝退到太平洋：

1. 先去飞书开发者后台建应用
2. 找 App ID 和 App Secret
3. 开事件、配置回调
4. 申请一堆权限
5. 回来本地继续配 OpenClaw

每一步单看都不难，但加在一起——对不起，我是来用 AI 的，不是来学开发的 😭

---

## 飞书官方插件带来了什么改变？

这次官方插件上线最大的变化就四个字：**门槛降了**。

具体说：

**扫码就能创建机器人**

以前：自己跑去飞书后台建应用
现在：安装命令跑起来 → 终端弹出二维码 → 扫一下 → 机器人自动建好

就问你跟注册个微信小程序似的，麻不简单？

**配置自动铺好了**

以前：自己找 App ID、Secret、事件、权限
现在：装完回后台一看——好家伙，App ID 有了，事件配好了，连权限都给你开了一部分！

**不只是能聊天了**

以前：只能发消息、回消息
现在：真的能"干活"了！

- 让它帮你写日程 → 自动写进飞书日历
- 让它查资料 → 自动生成飞书文档

这就不是"聊天机器人"了，这是"数字同事"！

---

##  内置飞书 vs 官方插件 怎么选？

| 场景 | 推荐 |
|------|------|
| 就想发个消息玩一玩 | 内置飞书够用了 |
| 认真当工作台用 | 官方插件更香 |
| 长期维护 | 官方插件（官方维护更稳定） |

---

## 普通人怎么装？4 步搞定

```bash
# 第 1 步：跑官方安装命令
npx -y https://sf3-cn.feishucdn.com/obj/open-platform-opendoc/3001f02e44c6269c6df09c01aad601cd_QWN3XUhUSQ.tgz install
```

然后：
- 扫码创建机器人 ✓
- 回后台核对 4 项（机器人、App ID、事件、权限）✓
- 飞书里发句话试试 ✓

全程不超过 10 分钟！
