## OpenClaw + Obsidian 是什么？

简而言之：

- **Obsidian** —— 本地化的 Markdown 知识库，你的第二大脑
- **OpenClaw** —— 7×24 小时待命的 AI 助理，接收指令后自动执行任务

传统内容生产流程：
1. 手动搜索并筛选素材
2. 复制粘贴并进行整理
3. 自行撰写成文

自动化后的流程：
1. 将链接发送给 OpenClaw
2. 自动抓取网页内容、提取要点、存入 Obsidian
3. 需要时直接调用

这相当于拥有一个全年无休的虚拟助手，处理所有繁琐的信息整理工作。

## 实际应用场景

### 1. 随手记录
发现优质文章后，直接通过飞书转发给 OpenClaw，系统自动提取核心要点并存储至知识库，全程无需手动操作。

### 2. 定时简报
设定每日清晨自动推送：天气信息 + 当日日程 + 关注的行业动态。

### 3. 知识库管理
每周定时让 OpenClaw 汇总整理近期笔记，自动生成周报，大幅提升复盘效率。

### 4. 自动化执行
仅需发出指令，剩余工作全部由 AI 完成。

## 核心配置：5 个关键文件

许多人误以为门槛很高，其实只需配置以下 5 个文件：

| 文件 | 作用 |
|------|------|
| **SOUL.md** | 定义 AI 性格与行为准则，确保输出风格符合预期 |
| **USER.md** | 记录用户身份、职业背景、偏好设置、时区等基本信息 |
| **HEARTBEAT.md** | 配置定时任务机制，实现自动化运维 |
| **AGENTS.md** | 制定工作流程规范，明确权限边界 |
| **MEMORY.md** | 建立长期记忆体系，跨会话保持信息连续性 |

完成这 5 个配置后，OpenClaw 将从"人工智障"真正转变为"智能助理"。

## 必备技能：4 个基础扩展

要让 OpenClaw 发挥完整能力，需要安装 4 个基础技能：

| 技能 | 功能 |
|------|------|
| **x-reader** | 解析公众号、小红书、B 站等国内平台链接 |
| **multi-search-engine** | 提供多引擎搜索能力 |
| **obsidian** | 直接读写 Obsidian 知识库 |
| **find-skills** | 发现并安装更多扩展技能 |

安装方式极其简单，每项仅需一条命令。
