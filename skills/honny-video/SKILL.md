# HonnyVideo Skill

使用 RunningHub Wan2.2 API 进行图生视频生成。

## 功能

| 功能 | 说明 |
|------|------|
| **图生视频** | 上传图片生成动态视频 |
| **进度轮询** | 每60秒查询任务进度 |
| **自动下载** | 任务完成后自动下载视频到本地 |
| **队列重试** | 并发满时自动等待重试 |
| **提示词优化** | 支持动作+表情+环境+运镜 |

## 配置

### 环境变量

在 `~/.openclaw/skills/honny-video/.env` 中配置：

```bash
# RunningHub API Key
RUNNINGHUB_API_KEY=你的API密钥

# 工作流 ID（图生视频-强一致性）
WORKFLOW_ID=2016727774387511298
```

### 获取 API Key

1. 访问 https://www.runninghub.cn
2. 登录 → 个人中心 → API
3. 创建 API Key

## 使用方法

### 提示词公式

```
(主体描述) + (表情动作) + (环境氛围) + (运镜方式)
```

**示例**：
- "一个女孩在玩耍"
- "看着镜头，比耶的手势，微笑，下雨天，镜头拉远"
- "可爱的小猫，伸懒腰，阳光明媚，镜头推进"

### 基本命令

```
生成一段视频
做一段图生视频
```

## 输出格式

```
🎬 视频生成成功！

📝 提示词：{使用的提示词}
🎥 视频：{视频URL}
💾 本地：{本地保存路径}

⏱️ 耗时：{时间}秒
💰 消耗：{金币}金币
```

## 新增功能详解

### 1. 进度轮询
- 提交任务后，每 **60 秒** 查询一次任务进度
- 只在进度变化时打印状态，避免刷屏

### 2. 自动下载
- 任务完成后自动下载视频到 `outputs/` 目录
- 文件命名格式：`video_{task_id}_{timestamp}.mp4`

### 3. 环境变量

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `RUNNINGHUB_API_KEY` | - | API 密钥（必填） |
| `WORKFLOW_ID` | 2016727774387511297 | 工作流 ID |
| `POLL_INTERVAL` | 60 | 轮询间隔（秒） |
| `MAX_RETRIES` | 5 | 提交重试次数 |
| `RETRY_DELAY_MS` | 60000 | 重试间隔（毫秒） |

---

*基于 RunningHub API v2*
